Semantic HTML elements

The main one:

Tag Name Description
<header> Header Specifies a header for a document or section
<nav> Navigation Defines a set of navigation links
<section> Section Defines a section in a document
<article> Article Specifies independent, self-contained content
<aside> Aside defines some content aside from the content it is placed in
<footer> Footer Specifies a footer for a document or section

Other:

Tag Name Description
<details> Details Defines additional details that the user can view or hide
<figcaption> Fig. caption Defines a caption for a <figure> element
<figure> Figure Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
<main> Main Specifies the main content of a document
<mark> Marked Defines marked/highlighted text
<summary> Summary Defines a visible heading for a <details> element
<time> Time Defines a date/time
